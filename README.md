# Qwen Graph Tester

Тестер графиков из ответов модели Qwen2.5-14B-Instruct-AWQ.

Страницы:
1. Ввод кода — вставляй полный ответ модели
2. Просмотр кода — смотри сохранённый код
3. Рендер графика — вставляй только код построения графика (plt.figure...plt.show)

Локальный запуск:
pip install -r requirements.txt
streamlit run app.py
